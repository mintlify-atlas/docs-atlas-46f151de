# Welcome to Aleo Network

Aleo is the first platform for building fully private applications using zero-knowledge proofs. With Aleo, developers can build decentralized applications that keep user data completely private while maintaining the security and transparency of a public blockchain.

## Why Aleo?

Aleo brings programmable privacy to the blockchain:

- **Zero-Knowledge Proofs**: Execute program logic while keeping inputs and outputs private
- **Full Privacy**: User data remains encrypted on-chain, visible only to authorized parties
- **Scalable Performance**: Decentralized proof generation enables high throughput
- **Developer Friendly**: Write privacy-preserving applications with Leo, a statically-typed programming language

## What You Can Build

Aleo enables a new category of privacy-preserving applications:

- **Private DeFi**: Trading, lending, and financial services with complete privacy
- **Confidential Identity**: Identity verification without revealing personal data
- **Private Gaming**: On-chain games with hidden state and fair play
- **Supply Chain**: Track provenance while protecting business secrets
- **Private DAOs**: Governance with confidential voting and proposals
